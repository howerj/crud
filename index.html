This is a simple query engine and logging system

In order to use this system you should either send a request to "/query" or to "/set". Use "?id=abc" with "/query" and "?id=abc&data=123" with "/set" to set a value. Any one can set a value, to query the data you should need authorization (either provided by the server.py script or an NGINX reverse proxy).
